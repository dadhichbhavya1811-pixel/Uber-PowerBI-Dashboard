# Uber-PowerBI-Dashboard
Developed an interactive Uber Analytics Dashboard in Power BI to analyze bookings, revenue, trip distance, cancellations, customer ratings, and payment methods. Used Power Query, DAX, and data modeling to create dynamic KPIs, visualizations, and actionable business insights.

# 🚖 Uber Analytics Dashboard | Power BI

An interactive **Power BI dashboard** built to analyze Uber ride data and generate actionable business insights. This project visualizes booking trends, revenue, trip performance, cancellations, customer ratings, and payment methods using dynamic KPIs and interactive reports.

---

## 📌 Project Overview

The **Uber Analytics Dashboard** helps businesses monitor key operational metrics and make data-driven decisions through interactive visualizations. It leverages Power Query for data transformation, DAX for calculations, and Power BI for reporting.

---

## 🎯 Objectives

- Analyze Uber booking patterns and ride demand.
- Monitor important business KPIs.
- Identify cancellation trends and their causes.
- Evaluate customer ratings and trip performance.
- Build an interactive dashboard for business decision-making.

---

## 📊 Dashboard Features

- 📌 Total Bookings
- 💰 Total Revenue
- 📏 Average Trip Distance
- ⏱️ Average Trip Time
- ❌ Cancellation Analysis
- ⭐ Customer Ratings
- 💳 Payment Method Distribution
- 🚗 Vehicle Type Analysis
- 📅 Booking Trends by Date & Time
- 🎛️ Interactive Slicers & Filters
- 📈 KPI Cards
- 🔍 Drill-through Analysis
