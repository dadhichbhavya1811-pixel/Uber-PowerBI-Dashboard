# Uber-PowerBI-Dashboard
Developed an interactive Uber Analytics Dashboard in Power BI to analyze bookings, revenue, trip distance, cancellations, customer ratings, and payment methods. Used Power Query, DAX, and data modeling to create dynamic KPIs, visualizations, and actionable business insights.

## Project Overview
This project analyzes Uber ride data using Power BI to provide business insights through interactive dashboards.

## Features
- Total Bookings
- Total Revenue
- Cancellation Analysis
- Trip Distance Analysis
- Payment Method Analysis
- Customer Ratings
- Time-based Analysis
- Interactive Filters and Slicers

## Tools Used
- Power BI
- Power Query
- DAX
- Excel
